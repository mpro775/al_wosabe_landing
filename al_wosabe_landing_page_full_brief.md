# ملف تنفيذ Landing Page احترافية لشركة الوصابي للتجارة

## 1. ملخص المشروع

المطلوب هو بناء موقع تعريفي/ليندنج بيج عصري واحترافي لشركة **الوصابي للتجارة – Al-Wosabe for Trading**، باللغتين العربية والإنجليزية، يعكس قوة الشركة وتاريخها وانتشارها في السوق اليمني، ويحوّل البروفايل القديم PDF إلى تجربة رقمية حديثة وجمالية ومتقنة.

الموقع يجب ألا يكون مجرد صفحة معلومات، بل يجب أن يعطي الزائر خلال أول ثوانٍ انطباعًا واضحًا بأن الشركة:

- شركة تجارية عريقة تأسست منذ عام 1986.
- تعمل في الاستيراد والتوزيع داخل اليمن.
- لديها فروع في مدن رئيسية.
- تمتلك أسطول توزيع يتجاوز 20 شاحنة.
- توفر منتجات مرتبطة بقطاع المركبات والدراجات مثل الإطارات، البطاريات، الزيوت، قطع غيار الدراجات، الإكسسوارات والأدوات.
- تتعامل مع مجموعة واسعة من العلامات التجارية المعروفة.

الهدف النهائي هو إنتاج موقع **Premium Modern Corporate Landing Page** يعكس الثقة، القوة، الحركة، الانتشار، والاحترافية.

---

## 2. الهدف التجاري من الموقع

### الهدف الأساسي

تحويل الزائر إلى تواصل مباشر مع الشركة أو طلب عرض سعر أو معرفة الفروع والمنتجات.

### الأهداف الفرعية

1. بناء ثقة فورية من خلال إبراز سنة التأسيس والخبرة.
2. توضيح مجالات نشاط الشركة بشكل بصري سريع.
3. عرض شبكة الفروع والتوزيع بطريقة احترافية.
4. إبراز العلامات التجارية التي تعمل معها الشركة دون مبالغة أو ادعاءات غير مثبتة.
5. تسهيل التواصل عبر الهاتف، واتساب، أو نموذج طلب عرض سعر.
6. توفير تجربة ثنائية اللغة عربية/إنجليزية باحترافية كاملة.
7. تحسين قابلية الظهور في محركات البحث عبر بنية SEO سليمة.

## 3. فهم هوية الشركة ونشاطها

### طبيعة العمل

شركة الوصابي للتجارة تعمل في:

- استيراد وتوزيع الإطارات.
- استيراد وتوزيع البطاريات.
- الزيوت ومواد التشحيم.
- قطع غيار الدراجات النارية.
- الإكسسوارات.
- الأدوات.
- توزيع منتجات وعلامات تجارية متعددة داخل السوق اليمني.

### التمركز المقترح للشركة على الموقع

يجب تقديم الشركة كمؤسسة تجارية موثوقة وليست كمحل بيع عادي.

الصياغة العامة:

> شركة يمنية عريقة في الاستيراد والتوزيع، تأسست منذ عام 1986، وتوفر منتجات موثوقة في الإطارات، البطاريات، الزيوت، قطع غيار الدراجات، الإكسسوارات والأدوات، عبر شبكة فروع وتوزيع تغطي مدنًا رئيسية في الجمهورية اليمنية.

### نقاط القوة الأساسية

| نقطة القوة | طريقة إبرازها في الموقع |
|---|---|
| التأسيس منذ 1986 | Hero + Stats + Timeline |
| الانتشار داخل اليمن | خريطة تفاعلية + Branches Section |
| أسطول التوزيع | Fleet Section + رقم 20+ |
| تنوع المنتجات | Product Categories Cards |
| العلامات التجارية | Brands Grid / Marquee |
| الثقة والخبرة | About + Mission + Values |

## 5. الهوية البصرية المقترحة

### الألوان

يتم استلهام الهوية من الكتالوج الحالي، لكن بتطويرها إلى شكل أكثر فخامة وحداثة.

```css
--color-primary: #FF8A00;
--color-primary-dark: #E87500;
--color-primary-light: #FFC247;

--color-dark: #1B1B1D;
--color-dark-soft: #252529;
--color-graphite: #343438;

--color-light: #F7F7F4;
--color-white: #FFFFFF;
--color-muted: #A7A7A7;

--gradient-primary: linear-gradient(135deg, #FF8A00 0%, #FFC247 100%);
--gradient-dark: linear-gradient(135deg, #1B1B1D 0%, #343438 100%);
```

### الاستخدام

- الخلفيات الداكنة للـ Hero و CTA وأجزاء القوة.
- البرتقالي للأزرار، الخطوط، الأرقام المهمة، والمؤثرات.
- الأبيض والرمادي الفاتح للمساحات الهادئة.
- يمكن استخدام أشكال مائلة مستوحاة من شريط الكتالوج، لكن بتصميم عصري.

### الخطوط

#### العربية

- Cairo
- Tajawal
- IBM Plex Sans Arabic

الأولوية المقترحة:

```css
font-family: 'Cairo', 'Tajawal', sans-serif;
```

#### الإنجليزية

- Inter
- Manrope
- Montserrat

الأولوية المقترحة:

```css
font-family: 'Inter', 'Manrope', sans-serif;
```

### العناصر البصرية

- خطوط إطارات Tire Tracks كخلفيات خفيفة.
- خطوط طريق Route Lines.
- زوايا مائلة Diagonal Shapes مستوحاة من الكتالوج.
- صور شاحنات ومخازن ومنتجات.
- خريطة اليمن بأسلوب Line Art.
- كروت بزجاجية خفيفة Glassmorphism عند الحاجة.
- ظلال ناعمة وليست مبالغًا فيها.

## 7.2 Hero Section

### الهدف

بناء الثقة والانطباع الفوري.

### التصميم

- خلفية داكنة فخمة.
- صورة رئيسية قوية: مبنى الشركة، شاحنات، مستودع، أو تركيب بصري يجمع أكثر من عنصر.
- شريط برتقالي مائل مستوحى من الكتالوج ولكن بشكل حديث.

- Badge واضح: Since 1986.

### النص العربي المقترح

```text
الوصابي للتجارة
شريك موثوق في التجارة والتوزيع منذ 1986

نوفّر منتجات موثوقة في الإطارات، البطاريات، الزيوت، قطع غيار الدراجات، الإكسسوارات والأدوات، عبر شبكة فروع وتوزيع تغطي مدنًا رئيسية في الجمهورية اليمنية.
```

### النص الإنجليزي المقترح

```text
Al-Wosabe for Trading
A Trusted Trading & Distribution Partner Since 1986

We supply reliable tires, batteries, lubricants, motorcycle spare parts, accessories, and tools through a strong branch and distribution network across Yemen.
```

### الأزرار

العربية:

- اطلب عرض سعر
- استعرض المنتجات

الإنجليزية:

- Request a Quote
- Explore Products

### الأنميشن

- العنوان يدخل من الأسفل مع Fade.
- الوصف يظهر بعد العنوان بـ delay بسيط.
- الأزرار تظهر بتأثير stagger.
- صورة Hero تتحرك Parallax بسيط عند تحريك الماوس أو التمرير.
- خطوط الإطارات في الخلفية تتحرك ببطء شديد.
- شارة Since 1986 تظهر بتأثير scale + opacity.

### ملاحظات مهمة

- لا تستخدم أنميشن مبالغ فيه.
- يجب احترام `prefers-reduced-motion`.
- يجب أن يبقى النص واضحًا ومقروءًا فوق الخلفية.

## 12. قواعد الأنميشن

### فلسفة الحركة

الحركة يجب أن تكون:

- ناعمة.
- فخمة.
- قليلة لكن مؤثرة.
- مرتبطة بالمعنى: توزيع، حركة، ثقة، نمو.

### لا نريد

- حركات كرتونية مبالغ فيها.
- سرعة عالية.
- عناصر تقفز بلا سبب.
- تحميل ثقيل.
- تأثيرات تسبب إزعاجًا على الموبايل.

### القواعد العامة

```ts
const motionConfig = {
  duration: 0.7,
  ease: [0.22, 1, 0.36, 1],
  stagger: 0.08,
}
```

### الحركات المطلوبة

| العنصر | الحركة |
|---|---|
| Hero title | Fade up + slight blur removal |
| Hero image | Slow parallax |
| Tire tracks | Very slow background movement |
| Stats | Count up + fade |
| Product cards | Stagger reveal + hover lift |
| Brand logos | Grayscale to color on hover |
| Map | SVG draw + city points pulse |
| Fleet | Number count + image reveal |
| CTA | Background route movement |

### Reduced Motion

يجب دعم:

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    scroll-behavior: auto !important;
  }
}
```

ولا تعتمد على الأنميشن لفهم المحتوى.
